# Pulse — Context-Aware Focus System

Next.js 15 + React 19 + TypeScript + Tailwind. Deploys to Vercel with zero config.

## Deploy

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/REPO.git
git push -u origin main
```

Then import on https://vercel.com/new — auto-detects Next.js.

## Local

```bash
npm install
npm run dev
```

Open http://localhost:3000
