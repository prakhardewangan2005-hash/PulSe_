# Pulse — Context-Aware Focus System

Built with **Vite + React + TypeScript**. Deploys to Vercel with zero config.

## Live
- Site: https://pulse-fawn-five.vercel.app
- Case study: https://pulse-fawn-five.vercel.app/work
- Figma: https://www.figma.com/proto/viW83K4WLdDSGYd3CtV2Lj/Pulse-%E2%80%94-Context-Aware-Focus-System--Community-

## Deploy

Push to GitHub, import on https://vercel.com/new. Vercel auto-detects Vite.

## Local

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Stack

- Vite 5
- React 18
- React Router 6
- TypeScript 5
